Krok 1

Nainstalujte si standardním způsobem modul Incomaker pro Magento2 pomocí Composeru:

composer require incomaker/magento2
bin/magento setup:upgrade
bin/magento setup:di:compile
bin/magento cache:flush

(musíte být ve složce, kde je nainstalováno Magento)

Krok 5

Pro připojení Incomakeru vyplňte položku API Key:

Jak získat API Key

Hodnotu Vašeho API Key naleznete ve svém Incomaker účtu po kliknutí na položku Eshop info.

Otevřete stránku my.incomaker.com

V pravém horním rohu klikněte na obrázek portrétu a zvolte možnost Eshop info

Zkopírujte hodnotu políčka API key

Zkopírujte API Key do Magenta a uložte. Pokud jste vložili správnou hodnotu, políčka Account ID a Plugin ID se doplní automaticky.

Připojení Magenta je nyní hotové a následuje automatická synchronizace dat.
