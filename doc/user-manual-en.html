Step 1

Install module Incomaker for Magento2 using standard procedure with Composer:

composer require incomaker/magento2
bin/magento setup:upgrade
bin/magento setup:di:compile
bin/magento cache:flush

(you must be in folder where Magento is installed for this to work)

Step 2

Log into you administrator account, open Stores section and select Configuration:

Step 3

In upper left corner of the screen, select configuration scope Main Website, alternatively select a specific website if you are running a multi-site installation:

Step 4

Expand item Incomaker:

Step 5

To connect with Incomaker, fill in value of API Key:

How to Obtain an API Key

You will get value of your API Key in your Incomaker account after clicking Eshop info.

Open my.incomaker.com

In upper right corner click on Eshop info

Copy value of field labeled as API key

Paste the value into Magento and save. If you entered correct value, Account ID and Plugin ID will be filled automatically.

Incomaker and Magento accounts are now connected and automatic data synchronization will follow.
